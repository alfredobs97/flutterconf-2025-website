## Overview

FlutterConf Spain is the official website for the conference that brings together the Flutter community in Spain. The site presents the event, its agenda, speakers, sponsors, community information, and contact details in a multilingual setup.

## Stack

- Next.js
- React 19
- TypeScript
- Tailwind CSS
- Firebase
- i18n in Spanish and English

## What it includes

- Homepage sections for introduction, information, and contact.
- Event agenda and content.
- Sections for speakers, sponsors, community, and code of conduct.
- Structure ready for multilingual content.

## Local development

```bash
npm install
npm run dev
```

The application runs at `http://localhost:9002`.

## Available scripts

- `npm run build`: generates the production build.
- `npm run start`: starts the generated build.
- `npm run typecheck`: runs TypeScript without emitting files.
